## 2. Описание предметной области

Система учета отзывов предназначена для компаний, желающих анализировать оценки о компании или товарах.Она учитывает текстовые отзывы, оценки, дату и время публикации, данные о клиенте. Она решает такие задачи, как: сбор и хранение отзывов, сортировка и фильтрация отзывов по заданным фильтрам,формирование графиков и отчетов, оповещение о негативных отзывах, осуществление поддержки обратной связи от имени компании.

## 8. Итоговый результат

**Реализованный функционал:**  
1.Создание отзывов через форму с полями:
текст отзыва (с валидацией на пустоту и запрещённые символы <, >, {, }, ;)
рейтинг (число от 1 до 5)
дата создания (в формате YYYY‑MM‑DD)
статус (new или old).
2.Валидация данных при отправке формы — проверка корректности всех полей и вывод ошибок пользователю.
3.Отображение списка отзывов в виде карточек с информацией:
текст
рейтинг
статус
дата создания
кнопка удаления
4.Управление отзывами:
показ всех отзывов
показ только отзывов со статусом new
сортировка по рейтингу (по убыванию)
удаление отдельных отзывов
5.Загрузка внешних данных из API 
6.Статистика по отзывам:
общее количество отзывов
сумму рейтингов
максимальный рейтинг
количество новых отзывов (status = "new")
7.Обратная связь с пользователем через сообщения о:
успешных операциях (добавление, удаление, загрузка и т. д.)
ошибках

**Потенциал развития:**  
1.Редактирование существующих отзывов — возможность изменять текст, рейтинг, дату или статус уже добавленных записей.
2.Фильтрация по дате — выбор отзывов за определённый период.
3.Фильтрация и сортировка по статусу — сортировка в порядке new → old или наоборот.
4.Поиск по тексту отзыва — поле для быстрого поиска по ключевым словам 
5.Пагинация — разбиение списка на страницы при большом количестве отзывов.
6.Визуализация статистики — графики и диаграммы 
7.Авторизация и разграничение прав
8.Дизайн страницы
9.Логирование действий — запись операций 
10.Поддержка разных языков — переключение языка интерфейса (русский, английский и т. д.).
